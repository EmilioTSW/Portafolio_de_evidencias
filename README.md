## Portafolio de Evidencias – HTML/CSS/JS
Fundamentos de Algébra / Tecnológico de Software

Este repositorio contiene el Portafolio de Evidencias desarrollado a lo largo del curso, integrando todas las actividades prácticas en un solo proyecto unificado, con diseño coherente y navegación intuitiva. 

## El portafolio está organizado de manera limpia y modular:

/
├── index.html              <-- Página principal (menú del portafolio)
├── css/
│   └── style.css           <-- Hoja de estilos global
├── js/
│   └── script_global.js    <-- Scripts globales (si se requieren)
├── assets/
│   └── images/             <-- Imágenes generales
├── actividades/            <-- Actividades integradas del curso
